PS中制作卡通按钮

需求​

制作一个卡通按钮,卡通风格，有渐变，有描边，有阴影

卡通休闲风 UI 按钮制作规范 (SOP)​

一、基础形体 (Base Shape)​

  • 工具：圆角矩形工具 (U)
  • 尺寸（九宫格极简版）：宽 80px × 高 80px（或 100px）
  • 圆角半径 (Radius)：20px
  • 初始填充色：纯蓝 #2D8BFF
  • 初始描边：无

二、图层样式配置表 (Layer Style Parameters)​

注意：务必按照以下从上到下的顺序叠加效果。

1. 渐变叠加 (Gradient Overlay) - 基础体积感​

  • Blend Mode (混合模式): Normal (正常)
  • Style (样式): Linear (线性)
  • Angle (角度): 90°
  • Color (渐变色): 底部 #2D8BFF → 顶部 #57A3FF

2. 内阴影 (Inner Shadow) 1 - 顶部高光边​

  • Blend Mode (混合模式): Screen (滤色)
  • Color (颜色): #FFFFFF (纯白)
  • Opacity (不透明度): 50%
  • Angle (角度): 90°
  • Distance (距离): 8px
  • Choke (阻塞): 0%
  • Size (大小): 0px (关键：保证硬边缘)

3. 内阴影 (Inner Shadow) 2 - 底部暗色边​

  • Blend Mode (混合模式): Normal (正常) 或 Multiply (正片叠底)
  • Color (颜色): #12438A (深紫蓝色)
  • Opacity (不透明度): 70%
  • Angle (角度): -90° (朝上挤压)
  • Distance (距离): 8px
  • Choke (阻塞): 0%
  • Size (大小): 0px

4. 描边 (Stroke) - 外围卡通线​

  • Size (大小): 8px
  • Position (位置): Outside (外部)
  • Blend Mode (混合模式): Normal (正常)
  • Color (颜色): #1A2A4A (极暗蓝色)

5. 投影 (Drop Shadow) - 底部硬阴影底座​

  • Blend Mode (混合模式): Normal (正常)
  • Color (颜色): #1A2A4A (与描边颜色一致)
  • Opacity (不透明度): 100%
  • Angle (角度): 90°
  • Distance (距离): 10px
  • Spread (扩展): 100% (关键：保证实心色块)
  • Size (大小): 0px (关键：保证硬边缘)

三、多色派生变量库 (Color Variations)​

派生新颜色只需修改以下三个参数，描边和投影保持 #1A2A4A 不变。

  • 红色按钮 (Red)

    • 渐变底：#FF3333 / 渐变顶：#FF7744
    • 暗边 (Inner Shadow 2)：#801010

  • 绿色按钮 (Green)

    • 渐变底：#22CC22 / 渐变顶：#77FF77
    • 暗边 (Inner Shadow 2)：#116611

  • 灰色按钮 (Grey / 设置按钮)

    • 渐变底：#58586A / 渐变顶：#7A7A8E
    • 暗边 (Inner Shadow 2)：#38384A

四、导出与工程化建议​

  • PS组织：右键图层组 Artboard from Group 转换为画板。
  • 尺寸调整：必须在右侧 Properties (属性) 面板中修改 W 和 H 输入框数值（切忌使用 Ctrl+T 拉伸，会破坏圆角）。
  • 导出操作：选中图层 → 右键 Export As... → PNG (勾选透明度)。
  • Unity 九宫格 (9-Slice) 设置：Sprite Editor 调整绿线边界：L:30, R:30, T:30, B:40。Image Type 设为 Sliced。
