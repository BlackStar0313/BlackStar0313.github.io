临界融合频率

在平时，人类会觉得蚊子或者苍蝇这类昆虫的反应速度很快，即便你的最快反应速度也不能轻松抓住他们，想捉住跑来跑去的猫狗也是个困难的事情。而相对于大象，人类却显得身手敏捷。处在同一个物理维度的生物，仅仅是存在物理上的快与慢的区别吗？但又为什么昆虫的生存时间很短？

一篇关于动物行为的论文里有关于动物在时间维度上是和人类一样长的。认为时间的认知是发自主观的体验，并不是客观存在的。也就是动物认为的时间刻度是不一样的，狗狗寿命只有十几年，但在狗的认知里他经历的时长是和人类相同的。也就是，狗的时间流逝速度比人类要快。 实验通过两个屏幕，通过不同的刷新频率闪光来观察动物行为。比较不同动物的CFF。

临界融合频率 （CFF）（Critical Flicker Frequency）

是指给予被试者一连串的闪光刺激，在闪光频率较低的时候，被试者可以感受到较明显的闪烁；当闪光的闪烁频率增高到一定水平时，被试者的闪烁感觉消失，从而产生稳定光亮的感觉，这个临界频率就叫做临界融合频率.

这个指标客观地反映了大脑神经处理视网膜接收信息的临界值，也侧面反映了不同动物感知时间快慢的能力

人类的CFF是60HZ，每秒闪60次就会没有闪的感觉。如果屏幕帧率降到30帧/s，就可以感到显示的不流畅，30帧/s以下的帧率就会感到卡顿。

狗的CFF是80HZ, 如果家里的电视刷新率是60Hz的，你会发现狗狗不是很爱看电视，人类感觉流畅但狗狗会感觉卡顿感到不舒服。

苍蝇的CFF是250HZ,他们感觉时间的速度是人类的4倍，这也是为什么人类打苍蝇很费劲，人类像是慢动作领域，而苍蝇时间在另一个维度里，但他们感受一生的时间是过了很久，比人类认为的时间要久。

结论指出，动物的体型和新陈代谢频率会影响动物的CFF。更小的体型，信号到达大脑的距离更短，更高的新陈代谢频率，信号会更快进行传递， 同理，人类个体之间也会存在时间认知不同， 单个个体的不同阶段也会存在相同情况，比如人类幼年时体型小且处于发育阶段新陈代谢快，感觉过一年的时间很漫长，成年后会感觉时间流逝越来越快。显得越来越稳重，其实是反应速度变慢了。

如果理论成立，在游戏中或许可以根据不同游戏主要用户的年龄分布情况，来对动效进行快慢的调整，比如中年人居多的游戏动效就多几帧长度，青年多的游戏动效少几帧就快点，幼儿游戏再加速。不根据游戏制作者的认知而是根据用户群的认知来制作会让玩家感受更舒服。

除了通过提升设备刷新率，内容的刷新率(影片或者游戏的制作帧率)，提升人类的观感，看起来舒服，也会通过一些技术手段做优化。比如在.net，MFC中，会通过双缓冲的策略去消除CFF对人类的影响。
