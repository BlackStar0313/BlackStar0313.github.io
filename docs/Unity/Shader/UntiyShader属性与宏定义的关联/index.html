Unity Shader 属性与宏定义的关联

属性声明与宏定义的关联​

在Unity中，属性声明与宏定义的关联是通过#pragma指令来实现的。

命名规则​

Toggle关键字命名规则:​

指定关键字名: [Toggle(KEYWORD_NAME)] 未指定时使用: _PROPERTY_NAME_ON

ToggleOff关键字命名规则:​

指定关键字名: [ToggleOff(KEYWORD_NAME)] 未指定时使用: _PROPERTY_NAME_OFF

KeywordEnum关键字命名规则:​

自动生成格式: _PROPERTY_NAME_ENUM_VALUE

比如

_Color_Color

示例​

Properties
{
    [KeywordEnum(Standard, Additive, Multiply, Screen, Overlay)]
    _BlendMode ("混合模式", Float) = 0
}

Pass
{
    # pragma multi_compile _BLENDMODE_STANDARD _BLENDMODE_ADDITIVE _BLENDMODE_MULTIPLY _BLENDMODE_SCREEN _BLENDMODE_OVERLAY
}

Unity会自动生成如下关联:

  • 属性名称: _BlendMode
  • 生成的宏前缀: BLENDMODE
  • 完整宏名称: 前缀+枚举值大写 _BLENDMODE_STANDARD, _BLENDMODE_ADDITIVE, _BLENDMODE_MULTIPLY, _BLENDMODE_SCREEN, _BLENDMODE_OVERLAY

工作流​

  1. 在Unity 编辑器 material 的 inspector 中 选择 属性
  2. 内部处理
  • 将_BlendMode 的属性设置成对应值。
  • 启用对应宏关键字。 例如 _BLENDMODE_STANDARD
  • 禁用其他所有宏关键字。
  1. 着色器编译,编译所有带有开启宏定义的变体。
  2. 执行条件代码.

注意点​

  • 命名必须严格匹配
  • 默认值为0
  • 着色器关键字（shader keywords） 在运行时动态切换, 数量有严格限制。
  1. 全局关键字(Global Keywords)
    • 限制：256个（在较早的Unity版本中是128个）
    • 作用范围：整个项目中所有着色器共享
  2. 局部关键字(Local Keywords)
    • 限制：每个着色器64个
    • 作用范围：仅限于定义它们的特定着色器

全局关键字 与 局部关键字 的区别​

特性全局关键字(Global Keywords)局部关键字(Local Keywords)
创建方式由multi_compile编译指令创建由shader_feature编译指令创建
作用范围在整个项目中共享同一命名空间仅在声明它们的着色器中可用
访问权限任何着色器都可以访问同名关键字可以在不同着色器中有不同含义
适用场景适合项目级设置（如：渲染管线特性、全局雾效开关等）适合材质级特性（如：纹理选项、特效开关等）
限制容易导致关键字空间冲突不会影响全局关键字限制

属性类型​

属性

根据文档 shader-keywords-material-inspector

MaterialPropertyDrawer

Unity内置的MaterialPropertyDrawer包括

  1. 值控制类型
  • ToggleDrawer ([Toggle]): 切换开关，用于启用/禁用着色器关键字
  • ToggleOffDrawer ([ToggleOff]): 反向切换，当开启时禁用关键字，关闭时启用关键字
  • KeywordEnumDrawer ([KeywordEnum]): 枚举选择器，允许从多个选项中选择一个并启用相应关键字
  • EnumDrawer ([Enum]): 显示下拉菜单，可使用Unity内置枚举或自定义名称/值对
  • PowerSliderDrawer ([PowerSlider]): 非线性响应滑块，用于Range属性
  • IntRangeDrawer ([IntRange]): 整数滑块，限制Range属性为整数值
  1. 装饰器类型
  • SpaceDecorator ([Space]): 在属性间添加垂直空间
  • HeaderDecorator ([Header]): 在属性前添加标题文本
  1. 其他类型
  • NoScaleOffset: 隐藏纹理的平铺和偏移控制
  • Normal: 标记纹理为法线贴图
  • HDR: 允许高动态范围颜色选择
  • Gamma: 在不同色彩空间之间正确转换值
  • MainTexture: 标记为主纹理
  • MainColor: 标记为主颜色
