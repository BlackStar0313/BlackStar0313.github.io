Unity URP 光照

1. 光照基本概念​

光照计算中的基本组成​

  1. 光源（Light Source）
  2. 表面（Surface）
  3. 观察者（Viewer）

光照基本类型​

  1. 环境光（Ambient Light）
  2. 漫反射（Diffuse）
    • 光线均匀散射到各个方向
    • 依赖表面法线和光线方向
    • 基于兰伯特定律： diffus = max(dot(normal, lightDir), 0);
  3. 镜面反射(Specular Reflection)
    • 特定角度产生的高光
    • 依赖视角，表面法线和光线方向
    • 常用 Phong或 Blinn-Phong模型

一些基础概念​

向量和法线​

  1. 法线向量 (Normal Vector)
    • 垂直于表面的向量
    • 决定表面朝向
    • 影响漫反射计算 float NdotL = max(dot(normal, lightDir), 0) 法线向量与光线向量的点乘， a点b = |a| * |b| * cos(theta) 得到两个向量的夹角cos值 这个余弦值直接反映了光线照射到表面的有效强度：
  • 当光线垂直照射表面时（θ = 0°），cos(θ) = 1，光照最强
  • 当光线与表面平行时（θ = 90°），cos(θ) = 0，光照为零
  • 当光线来自表面背后时（θ > 90°），cos(θ) < 0，理论上不会有光照结果越大，光照越强

向量点积公式 a点b = |a| * |b| * cos(theta) 得到两个向量的夹角cos值 向量叉积公式 a叉b = |a| * |b| * sin(theta) 叉积结果是一个向量，方向垂直于a和b

  1. 视线向量 (View Direction)
    • 从表面到观察者
    • 影响镜面反射计算
  2. 光线方向 (Light Direction)
    • 从光源到表面
    • 影响漫反射和镜面反射计算

材质属性​

  1. 反照率/Albedo
    • 表面反射各种波长光线的能力
    • 通常由纹理提供基础颜色
  2. 金属度（Metallic）
    • 表面是金属还是非金属
    • 金属会将镜面反射着色为其基础颜色
  3. 粗糙度/光滑度(Roughness/Smoothness)
    • 表面微观几何的粗糙程度
    • 影响镜面反射的锐度和分散度
  4. 法线贴图（Normal Map）
    • 模拟表面细节而不增加几何复杂度
    • 为每个像素提供独特的法线方向

2, 光照计算​

兰伯特Lambert 漫反射​

diffus = max(dot(normal, lightDir), 0);

半兰伯特 Lambert​

diffus = max(dot(normal, lightDir), 0) * 0.5 + 0.5;

冯氏反射模型 Phong​

反射向量(R) 和 视线向量(V) 的夹角 specular = pow(max(0, (dot(R, V))), shininess);

Blinn-Phong 模型​

半角向量(H) 和 法线向量(N) 的夹角 specular = pow(max(0, (dot(N, H))), shininess); 半角向量是光照方向(L)和视线方向(V)的平均值

#if defined(DEBUG_DISPLAY)
    half2   dynamicLightmapUV;
    half2   staticLightmapUV;
    float3  vertexSH;

    half3 brdfSpecular;

    // Mipmap Streaming Debug
    float2 uv;
    uint mipCount;

    // texelSize :
    // x = 1 / width
    // y = 1 / height
    // z = width
    // w = height
    float4 texelSize;

    // mipInfo :
    // x = quality settings minStreamingMipLevel
    // y = original mip count for texture
    // z = desired on screen mip level
    // w = loaded mip level
    float4 mipInfo;

    // streamInfo :
    // x = streaming priority
    // y = time stamp of the latest texture upload
    // z = streaming status
    // w = 0
    float4 streamInfo;

    float3 originalColor;

    float4 probeOcclusion;
#endif
};
