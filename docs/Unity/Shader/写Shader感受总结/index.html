写Shader感受总结

对于shader中加法与乘法混合的理解​

颜色本质是向量​

RGB三个分量

  • 红色(R) 范围 0-1
  • 绿色(G) 范围 0-1
  • 蓝色(B) 范围 0-1

乘法的直观感受​

  1. 过滤效果 比如通过一个透明玻璃，看一个红色的光。
  2. 反射特性 比如黄色的金属反射红色的光

加法的直观感受​

  1. 叠加效果 比如通过一个透明的玻璃，看一个红色的光和一个绿色的光。
  2. 发光效果 比如一个发光物体，发出红色的光和绿色的光。

混合/插值：过渡效果​

  1. 两种效果之间过渡 比如从阴影区域逐渐过渡到明亮区域
  2. 在卡通渲染中用于创建硬边界

使用情景​

使用乘法​

  1. 调制底色 当一种颜色影响另一种颜色
  • 光源颜色 x 材质颜色
  • 阴影颜色 x 材质颜色
  • 环境光遮蔽 x 环境光
  1. 减弱效果: 当需要按比例减弱到某个效果
  • 透明度 x 颜色
  • 光照衰减 x 光照强度

使用加法​

  1. 累计光源： 多个独立光源叠加
    • 主光源+额外光源+环境光
    • 漫反射+高光
  2. 叠加效果： 在基础效果上叠加额外效果
    • 基础颜色 + 自发光
    • 基础渲染 + 边缘发光

使用混合/插值​

  1. 状态过渡：两个状态之间平滑过渡
    • 阴影区域与明亮区域过渡
    • 透明过渡效果
  2. 多效果组合： 按权重组合多种效果
    • PBR中各种BRDF组合
    • 混合不同材质效果

一些效果的计算方式​

  1. 改变亮度
   result = colorSource + brightness;
  1. 改变对比度
  result = lerp(half3(0.5, 0.5, 0.5), colorSource, contrast + 1.0);
  1. 改变饱和度
  • 计算原理：公式 luminance = 0.299R + 0.587G + 0.114*B 将RGB颜色转换为感知亮度值，这个值与人眼实际感知到的亮度非常接近。
  half luminance = dot(colorSource, half3(0.299, 0.587, 0.114));
  result = lerp(luminance, colorSource, saturation);
  1. 改变色温 简单计算： （1,1,1） 暖色 （0,0,0） 冷色
  result = color * lerp(cool, warm, temperature * 0.5 + 0.5);
