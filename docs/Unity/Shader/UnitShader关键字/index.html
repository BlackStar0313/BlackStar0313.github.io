Unity Shader 关键字

Properties​

Properties块定义在Inspector中可以调整的属性

Properties
{
    _Color ("Color", Color) = (1,1,1,1)
	// ...........
}

SubShader​

SubShader块定义了Shader的渲染流程

SubShader
{
    Pass
    {
       //............
    }
}

Pass​

Pass块定义了Shader的渲染流程

Pass
{
    CGPROGRAM
    #pragma vertex vert
    #pragma fragment frag
    #include "UnityCG.cginc"
    ENDCG
	// .......
}

Tags​

Tags块定义了Shader的标签 RenderType 是Unity中常用的标签之一，用于指定渲染类型。 LightMode 是URP中常用的标签之一，用于指定光照模式。 Unity使用RenderType来对不同类型的物体进行分类和渲染 其他常见的RenderType值包括:

  • Opaque: 不透明物体 (墙体，地面， 普通物体)
  • Transparent: 透明物体 (玻璃，水， 树叶)
  • TransparentCutout: 透明度裁剪 完全不透明和完全透明 (草， 植物)
  • Background: 背景 (天空盒)
  • Overlay: 叠加层 (UI)
  • TreeOpaque: 树不透明
  • TreeTransparentCutout: 树透明度裁剪
  • TreeBillboard: 树 Billboard
  • Grass: 草
  • GrassBillboard: 草 Billboard

常见的光照模型 LightMode

Tags
{
    "RenderType"="Opaque"
}

Queue 是Unity中常用的标签之一，用于指定渲染顺序。 Unity常用队列值：

  • Background = 1000 (背景)
  • Geometry = 2000 (默认，不透明物体)
  • AlphaTest = 2450 (带Alpha测试的物体)
  • Transparent = 3000 (透明物体) 在不透明物体后渲染
  • Overlay = 4000 (叠加层，如UI)
Tags
{
    "Queue"="Transparent"
}

ZWrite​

ZWrite 透明物体通常设置ZWrite Off以避免遮挡后方透明物体

  • On = 写入深度缓冲区
  • Off = 不写入深度缓冲区
Pass
{
    ZWrite Off
}

Blend​

Blend 定义像素如何与一渲染像素混合 Blend SrcAlpha OneMinusSrcAlpha是最常用的透明混合 语法 : Blend SrcFactor DstFactor SrcFactor 源颜色混合因子 DstFactor 目标颜色混合因子

Pass
{
    Blend SrcAlpha OneMinusSrcAlpha
}

CGPROGRAM ENDCG​

CGPROGRAM块定义了Shader的顶点和片段着色器 ENDCG块结束CG程序

CGPROGRAM
ENDCG

_Time​

_Time Unity内置时间变量，用于获取当前时间

discard​

discard关键字用于丢弃当前像素, 停止片元处理，不会对帧缓冲区进行任何写入操作。片元丢弃

discard;

clip()​

clip()函数用于裁剪片元，如果片元在裁剪范围内，则保留，否则丢弃。

clip(dot(normal, viewDir) - 0.5);

frac()​

frac()函数用于返回浮点数的小数部分。

frac(x);

UnpackNormal()​

UnpackNormal()函数用于解压缩法线贴图。函数将颜色值转换为(-1,1)范围的法线向量

half3 normalTS = UnpackNormal(normal);

UniversalFragmentPBR​

UniversalFragmentPBR 是URP中用于实现PBR光照的函数。

half4 UniversalFragmentPBR(InputData inputData, SurfaceData surfaceData)

Cull​

Cull 用于指定剔除面，默认是Back，即背面剔除。

  • Cull Back - 背面剔除
  • Cull Front - 正面剔除 特殊效果：如外描边
  • Cull Off - 关闭剔除 渲染所有面 :如 树叶，布料，薄纸
Cull Back

数据类型 (UnityCG.cginc , Bulid-in 管线)​

  1. 基本值类型 | 类型 | 精度 | 范围 | 内存占用 | 适用场景 | |----------|---------|-----------------|---------|------------------------| | float | 32位 | ±1.5×10^−45 到 ±3.4×10^38 | 4字节 | 需要高精度的世界坐标、UV计算等 | | half | 16位 | ±6.0×10^−8 到 ±6.5×10^4 | 2字节 | 中等精度需求，如纹理坐标 | | fixed | 11位 | -2.0 到 +2.0 | ~1字节 | 颜色表示，范围有限但够用 |

  2. 向量类型 所有类型都有对应的向量类型

// 4维向量  (x, y, z, w)
float4 position;    // 32位每分量，适合位置
half4 direction;    // 16位每分量，适合方向
fixed4 color;       // 11位每分量，适合颜色

// 3维向量 (x,y,z)
float3 worldPos;    // 适合世界坐标
half3 normal;       // 适合法线方向
fixed3 albedo;      // 适合RGB颜色

// 2维向量 (x,y)
float2 uv;          // 适合UV坐标
half2 texOffset;    // 适合纹理偏移
fixed2 xy;          // 适合简单2D值

有多种访问方式

fixed4 color = fixed4(1.0, 0.0, 0.0, 1.0);
float r = color.r;
float g = color.g;
float b = color.b;
float a = color.a;

float x = color.x;
float y = color.y;
float z = color.z;
float w = color.w;

fixed2 rg = color.rg;
fixed3 rgb = color.rgb;
fixed4 rgba = color.rgba;
  1. 矩阵类型
// 4x4矩阵
float4x4 matrix;   // 4x4矩阵，适合变换矩阵
half3x3 rotationMatrix;  // 3x3矩阵，适合旋转矩阵
  1. 采样类型
// 2D纹理
sampler2D myTexture;
// 3D纹理
sampler3D my3DTexture;
// 立方体贴图采样
samplerCUBE myCubeMap;
  1. 整型
int myInt;  //32位整数
uint myUint; //32位无符号整数
  1. 布尔类型
bool myBool;  //1位布尔值

数据类型 (HLSL)​

  1. 基本值类型 | 类型 | 精度 | 范围 | 内存占用 | 适用场景 | |----------|---------|-----------------|---------|------------------------| | float | 32位 | ±1.5×10^−45 到 ±3.4×10^38 | 4字节 | 需要高精度的世界坐标、UV计算等 | | float2 | 32位 | 2个32位浮点数 | 8字节 | 2D向量，如位置、方向 | | float3 | 32位 | 3个32位浮点数 | 12字节 | 3D向量，如位置、方向 | | float4 | 32位 | 4个32位浮点数 | 16字节 | 4D向量，如位置、方向 | | float4x4 | 32位 | 16个32位浮点数 | 64字节 | 4x4矩阵，如变换矩阵 |

  2. 采样类型

TEXTURE2D(_MainTex);
SAMPLER(sampler_MainTex);

shader_feature​

shader_feature 用于定义可选的Shader变体, 宏定义

  1. 在材质检查器中创建可选择的功能
  2. 基于用户选择条件性编译代码
  3. 优化着色器，仅包含需要的代码变体
Pass
{
    # pragma shader_feature _FEATURE_NAME // 在材质检查器中创建可选择的功能
}

shader_feature与multi_compile很相似，但有重要区别： shader_feature:

  • 仅包含实际使用的变体
  • 不使用的变体不编译，节省内存
  • 适用于材质编辑器中的选项 multi_compile:
  • 编译所有可能的变体
  • 允许在运行时切换
  • 适用于全局设置或需要动态切换的功能

CBUFFER_START CBUFFER_END​

CBUFFER_START 和 CBUFFER_END 用于定义常量缓冲区(Constant Buffer)

  • 常量缓冲区用于存储Shader中常用的常量
  • 将相同材质属性打包在一起
  • 提高性能，减少寄存器压力,减少cpu向GPU传递数据次数
  • 避免在函数中频繁声明和使用
  • 统一内存布局，允许GPU多个材质快速切换
CBUFFER_START(UnityPerMaterial)
float4 _Color;
float _Intensity;
CBUFFER_END

SRP Batcher方式：

  • 材质属性组织在一个连续的内存块中
  • 相同着色器的材质共享绘制命令
  • 只更新常量缓冲区中的数据，不重新生成绘制命令

变换矩阵​

URP中常用的变换矩阵

  • unity_WorldToObject：世界空间到局部空间的变换

  • unity_MatrixVP：视图投影矩阵

  • unity_MatrixV：视图矩阵

  • unity_MatrixP：投影矩阵
