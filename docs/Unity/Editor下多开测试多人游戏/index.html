Editor下多开测试多人游戏

需求​

需要多人联机的游戏测试，需要多个客户端进行连接，一般情况是editor开一个游戏，然后打包mac/android或者其他的build出一个来，作为其他客户端测试，但是项目稍微大一点，build要很久，总不能每次修改一个代码就build一个真机包，那太慢了。

ParrelSync 是一个 Unity 插件，可以让你在 Editor 下多开测试多人游戏。

原理是clone当前unity项目，克隆目录，然后软连接方式共享文件。一些缓存文件是自己的，比如Library/,Temp/, Logs/这些目录是自己的，不会共享。

clone的一个实例就会在当前项目同级目录下生成各自实例的项目目录

使用​

  1. 安装 ParrelSync 插件

通过unitypackage直接引入 https://github.com/VeriorPies/ParrelSync/releases

通过upm 安装

https://github.com/VeriorPies/ParrelSync.git?path=/ParrelSync
  1. 创建副本

在ClonesManager窗口直接open in editor就会拉起这个副本的项目工程。

两个独立的unity工程，可以直接editor调试了。
