模式匹配

模式匹配是一种用于检查和处理数据结构的技术。C# 提供了多种模式匹配的方式，包括：

1. Null 检查​

可以使用模式匹配来优雅地处理可空类型：

int? maybe = 12;
if (maybe is int number)
{
    Console.WriteLine($"The nullable int 'maybe' has the value {number}");
}
else
{
    Console.WriteLine("The nullable int 'maybe' doesn't hold a value");
}

使用可空类型，可以检查一个值是否为 null。is表达式 可以检查一个值是否为某种类型，并将其赋值给一个变量。 也可以不赋值，直接使用 is 表达式。

int? maybe = 12;
if (maybe is int)
{
    var number = (int)maybe;
    Console.WriteLine($"The nullable int 'maybe' has the value {number}");
}

string? maybeString = "Hello, World!";
if (maybeString is not null)
{
    Console.WriteLine($"The nullable string 'maybeString' has the value {maybeString}");
}

注意：两种方式的区别

  • 不赋值方式需要显式转换，类型不匹配可能会抛异常
  • is 表达式赋值更安全，类型不匹配会返回 false

2. 类型测试​

用于测试变量与给定类型是否匹配：

public static T GetValue<T>(object obj)
{
    if (obj is IList<T> list)
    {
        return list[0];
    }
    return default;
}

3. 比较离散值​

使用 switch 表达式可以优雅地比较离散值：

public string GetWeather(string city)
{
    return city switch
    {
        "北京" => "晴天",
        "上海" => "多云",
        "广州" => "小雨",
        _ => "未知"
    };
}

4. 关系模式​

可以使用比较运算符测试数值：

public string GetWeather(int temperature)
{
    return temperature switch
    {
        < 10 => "非常冷",
        < 20 and > 10  => "有点冷",
        < 30 => "有点热",
        _ => "非常热"
    };
}

5. 多个输入​

可以同时测试多个条件：

public string GetWeather(int temperature, string city)
{
    return (temperature, city) switch
    {
        (< 10, "北京") => "非常冷",
        (< 20, "北京") => "有点冷",
        (< 30, "北京") => "有点热",
        (< 10, "上海") => "非常冷",
        (< 20, "上海") => "有点冷",
        (< 30, "上海") => "有点热",
        _ => "未知"
    };
}

6. 列表模式​

可以检查列表或数组中的元素，使用弃元 _ 匹配任意元素。

例如，对于以下交易记录：

04-01-2020, DEPOSIT,    Initial deposit,            2250.00
04-15-2020, DEPOSIT,    Refund,                      125.65
04-18-2020, DEPOSIT,    Paycheck,                    825.65
04-22-2020, WITHDRAWAL, Debit,           Groceries,  255.73
05-01-2020, WITHDRAWAL, #1102,           Rent, apt, 2100.00
05-02-2020, INTEREST,                                  0.65
05-07-2020, WITHDRAWAL, Debit,           Movies,      12.57
04-15-2020, FEE,                                       5.55

这是一个列表，想要匹配到一些元素，比如 DEPOSIT 和 WITHDRAWAL 的元素

public string GetTransactionType(string transaction)
{
    return transaction switch
    {
        [_, "DEPOSIT", .., var value] => $"DEPOSIT, {value}",
        [_, "WITHDRAWAL", .., var value] => $"WITHDRAWAL, {value}",
        _ => "UNKNOWN"
    };
}

// 输出
// DEPOSIT, 825.65
// WITHDRAWAL, 2100.00

这种方式类似于格式化字符串，但是可以匹配任意元素，而不仅仅是字符串。
