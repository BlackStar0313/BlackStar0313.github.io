Unity Shader 文档

语义（semantics）​

语义是Shader中的一种特殊变量，用于描述变量的用途。它在HLSL（High-Level Shader Language）中非常重要。

语义有以下几种：

  1. 输入语义：用于描述输入变量的用途。
// 输入语义
float4 vertex : POSITION;    // 顶点位置
float3 normal : NORMAL;      // 法线方向
float4 color : COLOR;        // 顶点颜色
float2 uv : TEXCOORD0;       // UV坐标
  1. 输出语义：用于描述输出变量的用途。
// 输出语义
float4 position : SV_POSITION; // 裁剪空间位置
float4 color : SV_Target;    // 输出到渲染目标的颜色
float4 somePosition : TEXCOORD0;     // 可以是任何位置数据
float2 uv : TEXCOORD1;               // 纹理坐标  可以传递任何类型的数据：位置、颜色、方向、自定义值等
float3 normal : TEXCOORD2;           // 法线数据
float3 tangent : TEXCOORD3;          // 切线数据
float4 color : COLOR;                // 顶点颜色
  1. 常量语义：用于描述常量的用途。
// 常量语义
float _Time : TIME; // 时间
float _SinTime : SIN_TIME; // 正弦时间
float _CosTime : COS_TIME; // 余弦时间
  1. 变量语义：用于描述变量的用途。
// 变量语义
float4 _Color; // 颜色
float _Intensity; // 强度
float _Float; // 浮点数
  1. 函数语义：用于描述函数的用途。
// 函数语义
float4 frag(float4 position : SV_POSITION) : SV_Target
{
    return position;
}
  1. 宏语义：用于描述宏的用途。
// 宏语义
#define PI 3.14159265358979323846
#define MAX_ITER 100

sv_ 前缀的语义是"系统值"语义（System Value Semantics），是特殊的保留语义，不能被用户重定义。 一个变量 ，必须有寓意标签，否则GPU不知道用途。

  • SV_POSITION 裁剪空间位置,告诉GPU这是顶点的最终投影后位置, GPU使用这个坐标进行裁剪，透视除法和屏幕映射
  • SV_TARGET 输出颜色,告诉GPU这是像素的最终颜色, GPU使用这个颜色进行渲染

内置纹理选项​

  • _MainTex ("Texture", 2D) = "white" {} // 纯白色纹理
  • _MainTex ("Texture", 2D) = "black" {} // 纯黑色纹理
  • _MainTex ("Texture", 2D) = "gray" {} // 灰色纹理
  • _MainTex ("Texture", 2D) = "bump" {} // 法线贴图
  • _MainTex ("Texture", 2D) = "red" {} // 红色纹理

内置矩阵​

  • unity_ObjectToWorld 对象空间到世界空间的矩阵 4x4的变换矩阵.
  • unity_WorldToObject 世界空间到对象空间的矩阵 4x4的变换矩阵.
  • unity_MatrixVP 视图投影矩阵 4x4的变换矩阵.
  • unity_MatrixV 视图矩阵 4x4的变换矩阵.
  • unity_MatrixP 投影矩阵 4x4的变换矩阵.
  • unity_MatrixInvVP 视图投影矩阵的逆矩阵 4x4的变换矩阵.
  • unity_MatrixInvVP 视图投影矩阵的逆矩阵 4x4的变换矩阵.

内置函数​

//将对象空间的位置转换为裁剪空间的位置。
float4 TransformObjectToHClip(float3 positionOS)

//将世界空间的位置转换为裁剪空间的位置。
float4 TransformWorldToHClip(float3 positionWS)
