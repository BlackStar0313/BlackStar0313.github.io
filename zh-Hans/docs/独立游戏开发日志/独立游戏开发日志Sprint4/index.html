独立游戏开发日志Sprint4

从这个sprint开始，终于开始做战斗demo了。

由于之前没有做过3D的TPS游戏，所以花了不到4天时间，看了3个TPS的示例项目，详细的拆解了一下TPS游戏他们每个细节是怎么做的。

  • Starter Assets - Third Person Character Controller Unity官方的一个教程，了解CharacterController ,Cinemachine ,InputSystem 这些基础组件是怎么使用的。
  • 3D Game Kit Lite Unity官方的一个示例，进一步了解整体战斗框架，包括角色控制，动画状态机，近战/远程武器使用，UI的Animation ,敌人AI的实现方式，关卡与机关的制作方式。
  • JU TPS 3 Unity AssetsStore上一个比较新维护，感觉比较全一点的TPS/FPS项目，花了60刀，拆解完相对完善的一套TPS战斗系统,包括武器系统，背包，射击，AI,角色控制。

每个实现都详细拆解了下，总的看下来，每一个的实现方式质量距上线差不少，大概了解了下别人是怎么实现的就好。感觉除了一些辅助的组件了解下有些价值，其他战斗系统业务逻辑跟之前做2d游戏的思路没什么太大差别。

其他时间

  • 前端框架对于ECS的使用方式进行了一点小优化。
  • 游戏UI做了横屏处理。
  • 动态双虚拟摇杆的基础逻辑实现。
  • 用ECS的方式加入了Character 可前后移动和旋转.

  • 开始学习Blender，做了个丑陋的小枪导到unity中。

  • 这种需要多点触控的，在editor里跑起来也不好测，打native包又太麻烦，unity的unity remote 5还挺好使，手机下app，连上电脑Editor Run 就能映射到手机上操作。虽然帧率低但是真方便。

以上，如有想法请多指教~ 感谢
