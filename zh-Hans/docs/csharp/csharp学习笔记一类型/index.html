类型

强类型 vs 弱类型​

每种编程语言都有自己的类型系统。一些脚本语言如 Python、Lua 等是弱类型语言，变量可以不指定类型直接赋值。

弱类型示例（JavaScript）​

let a = 1; // 数字类型
a = "123"; // 可以直接赋值为字符串，不会报错

强类型示例（C#）​

int a = 1;
a = "123";        // 编译错误：不能将string类型赋值给int

C#类型系统​

C#的类型系统分为值类型和引用类型。

值类型和引用类型的特点对比​

特点值类型引用类型
数据存储方式直接存储数据存储数据的引用（内存地址）
内存分配位置在栈上分配内存在堆上分配内存
赋值行为进行值复制复制引用
函数参数传递进行值复制复制引用

值类型示例代码​

int a = 1;
int b = a;        // b复制a的值
a = 2;
Console.WriteLine(b);    // 输出1：a和b是独立的变量

引用类型示例代码​

class Person {
    public string Name;
}

Person p1 = new Person();
p1.Name = "张三";
Person p2 = p1;         // p2引用p1指向的对象
p2.Name = "李四";
Console.WriteLine(p1.Name);  // 输出"李四"：p1和p2指向同一个对象

值类型详解​

所有值类型都继承自System.ValueType，后者继承自System.Object。

默认值规则​

所有值类型都隐式声明默认无参构造函数。对于所有简单类型，默认值都是 0 位模式生成值。

不同类型的默认值​

  • 数值类型
    • 整数类型（int、long 等）：0
    • 浮点类型（float、double）：0.0
    • decimal：0.0m
  • bool 类型：false（内部表示为 0）
  • char 类型：'\0'（空字符，ASCII 码为 0）
  • 枚举类型：值为 0 的枚举项
  • struct 类型：所有字段设置为对应类型的 0 位模式值
int a;            // 默认值为0
bool b;           // 默认值为false
float f;          // 默认值为0.0f
char c;           // 默认值为'\0'
DateTime date;    // 默认值为00:00:00.0000000, 1/1/0001

值类型分类​

1. 结构类型（struct）​

包含：

  • 常量
  • 字段
  • 属性
  • 方法
  • 事件
  • 索引器
  • 运算符
  • 构造函数
  • 析构函数
  • 嵌套类型

5. decimal 类型​

double d = 0.1;
double sum1 = d + d + d;     // 0.30000000000000004

decimal m = 0.1m;
decimal sum2 = m + m + m;    // 0.3

6. 布尔类型​

类型说明位数取值范围
bool布尔类型1 位true/false

注意：bool 类型只有 true 和 false 两个值，不能用 0 和 1 或 null 表示。

7. 枚举类型​

每个枚举类型都有一个基础类型（byte、sbyte、short、ushort、int、uint、long 或 ulong）。

8. 元组类型​

(int, string) pair = (1, "One");
Console.WriteLine(pair.Item1);    // 输出1
Console.WriteLine(pair.Item2);    // 输出One

var tuple = (1, "Hello", 3.14);
Console.WriteLine(tuple.Item1);   // 输出1
Console.WriteLine(tuple.Item2);   // 输出Hello
Console.WriteLine(tuple.Item3);   // 输出3.14

9. 可空类型（Nullable<T>）​

int? nullableInt = null;
Console.WriteLine(nullableInt.HasValue);    // 输出False
Console.WriteLine(nullableInt.Value);       // 抛出异常

10. 结构类型示例​

struct Point {
    public int X;
    public int Y;
}

值类型的装箱和拆箱​

int a = 1;
object b = a;         // 装箱
int c = (int)b;      // 拆箱

装箱过程：

  1. 在堆上分配内存
  2. 将值类型的值复制到新内存
  3. 返回对象引用

拆箱过程：

  1. 检查对象类型
  2. 将值复制到值类型变量
  3. 返回值

引用类型详解​

引用类型包括：

  1. 类类型
  2. 对象类型（Object 是所有类型的基类）
  3. 动态类型（dynamic）
  4. 字符串类型
  5. 接口类型
  6. 数组类型
  7. 委托类型
  8. 表达式类型（用于 lambda 表达式）
